Installation#

Isaac Lab Arena supports installation from source inside a Docker container.

  1. Clone the repository and initialize submodules:

git clone git@github.com:isaac-sim/isaac_arena.git

git submodule update --init --recursive

  1. Launch the docker container:

./docker/run_docker.sh

  1. Optionally verify installation by running tests:

pytest -s isaac_arena/tests/

With isaac_arena installed and the docker running, you’re ready to build your first IsaacLab-Arena Environment. See First Arena Environment to get started.
